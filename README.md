# Mark AI for Field Marketing

A self-contained HTML prompt explorer for field marketers. It includes 26 research prompts, a featured five-prompt introduction, use-case filters, prompt viewing, raw copy, prompt customisation, and local progress tracking.

## Files

- `index.html` — the complete interactive experience. No build step required.
- `.nojekyll` — keeps GitHub Pages from applying Jekyll processing.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `README.md`, and `.nojekyll` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your main branch and `/root`.
6. Save. GitHub will provide the published URL once the page is deployed.

## Notes

The app stores “Mark tried” progress locally in the user’s browser. It does not send data anywhere.
